# Fortex36 hour project

This repository is a work of 36 hours brain stroming and implementing with a purpose of monitoring health of person, generating valid reports and alerting stakeholders in case of adversary

The report consists of many mini projects or microservice apps that acheive unique things

- The main app is the Mitra health (medflow-erp app)
- The chatbot-ai is specially designed to address common persons issue. It does by running a ollam model on locally hosted AI
- The radianceai-x-ray-enhancer microservice is used to enhance xray and analyze thee enhanced image using MedGemmaAI, it is also hosted locally pusruably as a  edge computing
- The sentinel-notification-system used to send message manually by a nurse to the respective stakeholders for that individual
- The cariovision-ai is a app that uses ecg lab report by analyizing the report with both custom algorithms and AI
- The safetynet-iot-monitoring is a app that uses 5 differnet agents to grade a person's health
