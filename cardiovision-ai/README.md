## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`

# cardiovision-ai

## ECG Interpretation

- to run the app go to the cardiovision-ai webapp run the following command in bash or terminal or command prompt

```cmd
npm run dev
```

- Upload your ecg report and get the fully analyzed report by clicking the report button on pdf
- sometimes you may get the google_api_key is not valid as this may be due to expired key use a new one instead
